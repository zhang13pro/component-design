# Vue3 组件库

## 项目规范

### git 提交

- feat: 新功能
- fix: 修改 bug
- docs: 文档
- perf: 性能相关
- refactor: 代码重构（就是不影响使用，内部结构的调整）
- test: 测试用例
- style: 样式修改
- workflow: 工作流
- build: 项目打包构建相关的配置修改
- ci: 持续集成相关
- revert: 恢复上一次提交（回滚）
- wip: work in progress 工作中 还没完成
- chore: 其他修改（不在上述类型中的修改）
- release: 发版
- deps: 依赖相关的修改

## 项目难点

- 组件的样式管理
- 如何实现布局组件
- 表单组件
